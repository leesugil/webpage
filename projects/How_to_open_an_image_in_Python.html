How to open an image in Python¶

There are many ways to load image files to Python. In this note, we introduce two packages to open images in Python that are often used in data science:

  • matplotlib.pyplot
  • keras.preprocessing

This note also shows whether we use / or \ when indicating a subdirectory in Python. This could be very confusing as MS-DOS uses the inverted slash \ to indicate folders, whereas almost all other systems including Python and Unix-based systems use the forward slash /.

Using Matplotlib¶

This is perhaps the simplest way for the data science purpose because you would import matplotlib anyway for visualizing your data.

Matplotlib imports image as a numpy array directly with the indices [row, colomn, RGB].

In [1]:
# Import images in matplotlib.pyplot
import numpy as np
import matplotlib.pyplot as plt

img = plt.imread('images/sugil.jpg')
print(type(img))
print(img.shape)
plt.imshow(img)

Sometimes one wishes to consider a particular color channel only. This can be achieved as:

In [2]:
img_red = img[:, :, 0]
plt.imshow(img_red, 'gray')
Out[2]:
<matplotlib.image.AxesImage at 0x1b117a4de08>

For most of the time in data science (especially in neural network), having a row and a column index of an image data is degarded as redundant. Thus we often reshape the image array into a vector form of RGB values,

In [3]:
img_vector = img.reshape(img.shape[0]*img.shape[1], img.shape[2])
plt.imshow(img_vector.reshape(img.shape[0], img.shape[1], img.shape[2]))
Out[3]:
<matplotlib.image.AxesImage at 0x1b117ad0208>

Using Keras preprocessing.image¶

Keras offers many preprocessing tools, and Python Image Library (PIL) is integrated in Keras. PIL loads an image as a PIL image file rather than as a numpy array, so we have to use the image.img_to_array() function to convert the loaded image into a numpy array. Despite the inconvenience, PIL offers many useful functions to apply basic transformation to the image.

In [4]:
# Opening an image in Keras
from keras.preprocessing import image
import matplotlib.pyplot as plt

# Load the raw image file
img = image.load_img('images/sugil.jpg')
plt.imshow(img)
type(img)

Using TensorFlow backend.
Out[4]:
PIL.JpegImagePlugin.JpegImageFile

Because img has a PIL image data structure, running img opens the image right away, though we used plt.show() to compress the data and save some time.

Here's how to resize the image.

In [5]:
img_square = image.load_img('images/sugil.jpg', target_size = (224, 224))
plt.imshow(img_square)
Out[5]:
<matplotlib.image.AxesImage at 0x1b124dc6d08>

Finally, here is how to convert a PIL image data to a numpy array.

In [6]:
x = image.img_to_array(img)
print(x)
print(type(x))
