How to make 3D plots rotate in Jupyter notebook¶

When you plot a 3D data in Jupyter notebook, you might wish to rotate the figure, otherwise we are losing the advantage of having 3D graphics. However, this is not achieved by default, so here's a short note on how to enable it.

Basically we use the magic command %.

(Making figures interactable on the web is a different thing, check out here.)

In [1]:
# Make 3D plots rotate by using the magic command (the percent sign)

%matplotlib notebook

import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

figure = plt.figure()
axes = figure.gca(projection = '3d')

def f(a, b):
    return np.sin(np.sqrt(a**2 + b**2))

x = np.arange(-5, 5, 0.25)
y = np.arange(-5, 5, 0.5)
x, y = np.meshgrid(x, y)
z = f(x, y)

#axes.plot_surface(x, y, z)
axes.plot_wireframe(x, y, z)
